# petguard-vision-ia-SANCHEZDEPAULA

## Objetivo
Construir un prototipo que, dado una imagen de un animal, utiliza un modelo preentrenado de Hugging Face para identificar la especie (clasificación de imágenes).

## Estructura del repositorio
- `/src` : Código fuente (script `classify.py`).
- `/assets` : Imágenes utilizadas para pruebas.
- `README.md` : Documentación del proyecto.
- `requirements.txt` : Dependencias.

## Instrucciones de ejecución
1. Clona o descarga el repositorio.
2. Crea y activa un entorno (recomendado con Anaconda):
   ```bash
   conda create -n petguard python=3.10 -y
   conda activate petguard
